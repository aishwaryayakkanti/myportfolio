# Professional Portfolio Website

A modern, responsive, and interactive personal portfolio website showcasing software development skills, professional journey, certifications, and technical projects.

## Visuals & Features

*   **Interactive Terminal Intro**: An animated command-line greeting interface on the landing page.
*   **Futuristic Design & Aesthetics**: Built using curated neon color schemes, glassmorphic UI elements, high-tech grid/circuit patterns, and smooth scroll animations.
*   **Dynamic Project Filtering**: Displays coding projects categorized by domain (Java, Python, DevOps, AI/ML, Web) with instant filtering, modal details, and direct GitHub links.
*   **Interactive Timeline**: A responsive experience timeline showcasing professional history, key achievements, and internship roles.
*   **Credential Showcase**: Structured view of professional certifications with interactive categorization.
*   **Theme Toggle**: Full light and dark mode ambient display modes.

## Tech Stack

*   **Core**: React, TypeScript, Vite
*   **Styling & UI**: Tailwind CSS, CSS Grid, shadcn/ui, Lucide Icons
*   **Routing & API**: React Router DOM, TanStack Query

## Local Setup

To run the application locally, ensure you have Node.js installed, then follow these steps:

1. Install dependencies:
    ```sh
    npm install
    ```

2. Start the local development server:
    ```sh
    npm run dev
    ```

3. Open your browser and navigate to the address shown (default is `http://localhost:8080` or `http://localhost:8081`).
